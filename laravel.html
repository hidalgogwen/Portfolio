TIMELY FILMS REVIEW

During our laboratory in ITELEC 4100, we made a movie review website using Laravel. This website has 3 modules: Admin, User, and Visitor. It includes CRUD, authentication for user type, adding movie ratings, search and filtering of data. It uses MySQL as its database management. I serves as the full stack developer for this project.
